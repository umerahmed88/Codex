# Repository guide

This repo contains **`arabic-app/`** — a native (Expo/React Native) Arabic
self-improvement app. That is the active project.

**To work on it, read `arabic-app/HANDOFF.md` first** — it captures the full
project state: all 10 build phases are complete; what remains is the manual
setup (Supabase, RevenueCat, store accounts) listed there. All work is on
branch `claude/new-repo-folder-u8r4th`; PRs merge to `main`.

Other top-level files (`docs/`, the Surah document) are unrelated to the app.
